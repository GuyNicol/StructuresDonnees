Les principaux tris simples expliqués sont :

* Le tri à bulles (Bubble Sort)
* Le tri par sélection (Selection Sort)
* Le tri par insertion (Insertion Sort)

D'autres tris simples sont expérimentés dans les exercices :

* Le tri cocktail
* Le tri impair-pair
